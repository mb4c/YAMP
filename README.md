# YAMP
YAMP is a cross-platform music player written in C++.

<img src="YAMP.png">

# Features
- Supports most audio formats
- Playlists
- Themes
- Unicode support

# Planned Features
- Customizable hotkeys
- Lyrics

# Building

#### Clone repository
```
git clone https://github.com/mb4c/YAMP
``` 

#### Linux

```
mkdir build
cd build
cmake ..
make
```
